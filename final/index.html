On Hover

Triggers

When user hovers over image/video

Rules

Cursor changes to pointer

Image is enlarged

Inner drop shadow fades in

Info text for show fades in

Play button and add button fades in

Image stays enlarged until user exits hover

Switches from static image to video trailer after a few seconds

Video trailer of show plays

When user exits hover, video reverts back to image

When user exits hover, image goes back to original size

Inner drop shadow stays as long as user is hovering

When user exits hover, drop shadow and text fades out
